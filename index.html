    graphs

    graphs is simple graph library for Scala and Scala.js

    Features

    • Simple graph creation API
    • Depth-first traversal / search
    • Breadth-first traversal / search
    • Topological sorting
    • Dijkstras algorithm for shortest paths
    • GraphML import / export (experimental)
    • force based layout (planned)
    • cross compiled for Scala 2.11, 2.12, Scala.js

    Alternatives

    graphs is similar to Graph for Scala but not aiming to model every kind of graph, making the core classes smaller and maybe easier to understand. If you need more than simple directed / undirected graphs please consider using that instead of graphs, be aware that Graph for Scala does not have Scala.js support currently.
